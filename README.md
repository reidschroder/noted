# Noted


## Purpose
Note taking application that saves your notes to a json file.

When clicking on an old note title from the left hand side, the note will then show up on the page.

## Built With
* HTML
* CSS
* JavaScript
* Node.JS
* Express.JS
* Deployed to Heroku
